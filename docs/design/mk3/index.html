5. State grammar

Every view has clear populated, empty, partial, degraded, and critical states

live SSE connected

New conversations and messages highlight, settle, and preserve scroll. Status chip shows last tick.

stale fallback polling

SSE unavailable; ETag polling takes over. Views show snapshot age.

critical schema mismatch

HTTP/reader remains visible; ingest disabled; health explains runtime/database version conflict.

partial derived stale

Insights/cost/topology panels are present but labeled stale, partial, rebuilding, or unavailable.

paste boundaries unknown

Message flagged as paste but no spans; show whole-message fallback and disable typed-only copy.

attachment missing

Provider IDs and refs remain visible; preview/open disabled with exact missing reason.

topology unresolved

Native parent preserved as ghost node; late repair can resolve when parent arrives.

mutation pending

Marks/notes/saved views use optimistic state with deterministic no-op/error outcomes.

empty no archive

Command center shows first-run setup and ingest affordances, not blank search.

auth token required

Do not render raw/local path details until authenticated.

compare no alignment

Fallback to timestamp/order alignment and label confidence.

unsafe raw HTML

Render text-only or sandboxed strict-CSP iframe; never trusted main DOM.
