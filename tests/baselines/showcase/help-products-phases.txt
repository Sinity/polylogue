Usage: python -m polylogue products phases [OPTIONS]

  List durable session-phase products.

Options:
  --conversation-id TEXT  Only phases from one conversation
  --provider TEXT         Limit to provider
  --since TEXT            Only rows on/after this timestamp
  --until TEXT            Only rows on/before this timestamp
  --kind TEXT             Only this session phase kind
  --limit INTEGER         Maximum rows  [default: 50]
  --offset INTEGER        Start offset  [default: 0]
  --json                  Output as JSON
  -h, --help              Show this message and exit.
