Usage: polylogue ops embed backfill [OPTIONS]

  Run the first embedding batch with per-session cost feedback.

  Prints the cost preflight, confirms, then iterates pending sessions (or all
  sessions if ``--rebuild``) and emits running totals so the user can
  interrupt before the soft monthly cap kicks in.

Options:
  --max-sessions INTEGER RANGE    Maximum sessions to process in this catch-up
                                  window.  [x>=1]
  --max-messages INTEGER RANGE    Maximum messages to include in this catch-up
                                  window.  [x>=1]
  --max-cost-usd FLOAT RANGE      Approximate maximum Voyage cost to include
                                  in this catch-up window.  [x>0.0]
  --stop-after-seconds INTEGER RANGE
                                  Stop starting new sessions after this many
                                  seconds.  [x>=1]
  --max-errors INTEGER RANGE      Stop the catch-up window after this many
                                  provider errors.  [x>=1]
  --rebuild                       Re-embed every session, not just the pending
                                  ones.
  -y, --yes                       Skip the cost confirmation prompt.
  -h, --help                      Show this message and exit.
