Usage: polylogue ops debt list [OPTIONS]

  List archive debt across assertions, tiers, convergence, embedding, and FTS
  signals.

Options:
  --kind [archive-tier|assertion-candidate|convergence|embedding|fts]
                                  Restrict rows to one debt kind. Repeatable.
  --only-actionable               Only show rows with a direct operator
                                  action.
  --limit INTEGER                 Maximum rows to emit.
  --exact-fts                     Run exact FTS reconciliation counts.
  -f, --format [text|json]        Output format.
  -h, --help                      Show this message and exit.
