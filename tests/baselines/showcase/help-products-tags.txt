Usage: python -m polylogue products tags [OPTIONS]

  List durable session-tag rollup products.

Options:
  --provider TEXT   Limit to provider
  --since TEXT      Only rows on/after this timestamp
  --until TEXT      Only rows on/before this timestamp
  --query TEXT      Substring match against the tag name
  --limit INTEGER   Maximum rows  [default: 100]
  --offset INTEGER  Start offset  [default: 0]
  --json            Output as JSON
  -h, --help        Show this message and exit.
