Usage: polylogue ops insights coverage [OPTIONS]

  List archive coverage buckets by origin, day, or week.

Options:
  --group-by [origin|day|week]  Bucket coverage by origin, day, or ISO week
                                [default: origin]
  -o, --origin TEXT             Only this origin
  --since TEXT                  Only buckets at/after this timestamp or date
  --until TEXT                  Only buckets at/before this timestamp or date
  -l, --limit INTEGER           Maximum rows  [default: 50]
  --offset INTEGER              Start offset  [default: 0]
  -f, --format [json]           Output format
  -h, --help                    Show this message and exit.
