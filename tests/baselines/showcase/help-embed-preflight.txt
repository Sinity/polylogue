Usage: polylogue embed preflight [OPTIONS]

  Estimate token count and Voyage cost for the pending backlog.

  Read-only — does not touch the embedding provider. Use this before ``embed
  enable`` or ``embed backfill`` to budget the first run.

Options:
  --max-sessions INTEGER RANGE  Estimate only the next bounded session window.
                                [x>=1]
  --max-messages INTEGER RANGE  Estimate only the next bounded message window.
                                [x>=1]
  --max-cost-usd FLOAT RANGE    Estimate only the next approximate cost
                                window.  [x>0.0]
  --rebuild                     Estimate re-embedding every session, not just
                                pending ones.
  -f, --format [text|json]      Output format.
  -h, --help                    Show this message and exit.
