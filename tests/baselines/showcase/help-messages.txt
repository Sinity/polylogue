Usage: polylogue messages [OPTIONS] [SESSION_ID]

  Show paginated messages for a session.

Options:
  -r, --message-role TEXT         Filter by message role
  --message-type TYPE             Filter by message content type
  -l, -n, --limit INTEGER         Max messages to return
  --offset INTEGER                Offset for pagination
  --no-code-blocks                Exclude code blocks
  --no-tool-calls                 Exclude tool calls
  --no-tool-outputs               Exclude tool outputs
  --no-file-reads                 Exclude file reads
  --prose-only                    Show only prose text
  -f, --format [markdown|json|plaintext]
                                  Output format
  --help                          Show this message and exit.
