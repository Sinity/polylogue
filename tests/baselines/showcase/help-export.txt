Usage: polylogue export [OPTIONS] [SESSION_ID]

  Export one known session by ID. IDs can use the origin:id-prefix form (e.g.
  claude-ai-export:abc123).

  The session id may be omitted when a root filter like ``--latest`` or
  ``--origin`` narrows the archive to a single match (#1642).

Options:
  -f, --format [markdown|json|html|yaml|plaintext|csv|obsidian|org]
                                  Output format  [default: markdown]
  --fields TEXT                   Fields for JSON/YAML outputs
  -h, --help                      Show this message and exit.
