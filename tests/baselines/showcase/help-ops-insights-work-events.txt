Usage: polylogue ops insights work-events [OPTIONS]

  List durable work-event insights.

Options:
  --session-id TEXT          Only events from one session
  --session-date-since TEXT  Only events whose canonical session date is
                             on/after this date
  --session-date-until TEXT  Only events whose canonical session date is
                             on/before this date
  --heuristic-label TEXT     Only this weak heuristic work-event label
  --query TEXT               FTS query against insight search text
  -l, --limit INTEGER        Maximum rows  [default: 50]
  --offset INTEGER           Start offset  [default: 0]
  -f, --format [json]        Output format
  -h, --help                 Show this message and exit.
