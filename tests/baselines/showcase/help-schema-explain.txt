Usage: polylogue ops schema explain [OPTIONS]

  Explain a package element schema with evidence and annotations.

Options:
  -p, --provider TEXT  Provider name  [required]
  --version TEXT       Schema version (default: latest)
  --element TEXT       Element kind inside the package
  -f, --format [json]  Output format
  -v, --verbose        Show semantic roles and coverage
  --review-evidence    Show evidence for role assignment decisions
  -h, --help           Show this message and exit.
