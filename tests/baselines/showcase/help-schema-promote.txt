Usage: polylogue schema promote [OPTIONS]

  Promote an evidence cluster to a new registered package version.

Options:
  --provider TEXT        Provider name  [required]
  --cluster TEXT         Evidence cluster ID to promote  [required]
  --with-samples         Re-load samples for full schema generation
  --max-samples INTEGER  Max samples when using --with-samples
  --json                 Output as JSON
  -h, --help             Show this message and exit.
