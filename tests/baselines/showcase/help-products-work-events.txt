Usage: python -m polylogue products work-events [OPTIONS]

  List durable work-event products.

Options:
  --conversation-id TEXT  Only events from one conversation
  --provider TEXT         Limit to provider
  --since TEXT            Only rows on/after this timestamp
  --until TEXT            Only rows on/before this timestamp
  --kind TEXT             Only this work-event kind
  --query TEXT            FTS query against product search text
  --limit INTEGER         Maximum rows  [default: 50]
  --offset INTEGER        Start offset  [default: 0]
  --json                  Output as JSON
  -h, --help              Show this message and exit.
