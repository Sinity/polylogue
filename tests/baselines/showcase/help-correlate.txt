Usage: polylogue correlate [OPTIONS] SESSION_ID

  Show git commits and GitHub refs within the session's time window (#1690).

Options:
  -r, --repo-path TEXT            Path to git repository. Defaults to current
                                  directory.
  -w, --since-hours INTEGER       Hours before/after session to scan for
                                  commits.
  -f, --format [json]
  --confidence-threshold FLOAT    Minimum confidence for file-overlap commit
                                  detection.
  --github-api / --no-github-api  Cross-reference issue/PR references with
                                  GitHub API if gh CLI is available.
  --otlp                          Add OTLP span evidence to the correlation
                                  output.
  -h, --help                      Show this message and exit.
