Usage: polylogue demo seed [OPTIONS]

  Create a ready-to-query deterministic demo archive.

Options:
  --root PATH                Archive root to seed. Defaults to
                             POLYLOGUE_ARCHIVE_ROOT.
  --force                    Replace the generated demo source directory
                             before seeding.
  --with-overlays            Seed deterministic user overlays after archive
                             ingest.
  -f, --format [plain|json]  [default: plain]
  -h, --help                 Show this message and exit.
