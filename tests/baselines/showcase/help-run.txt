Usage: python -m polylogue run [OPTIONS]

  Run pipeline stages on configured sources.

Options:
  --preview                       Preview work without writing
  --stage [acquire|parse|materialize|render|index|generate-schemas|all]
                                  Pipeline stage: acquire (store raw), parse
                                  (extract conversations), materialize
                                  (derived products), render (output), index
                                  (search), generate-schemas, or all
                                  [default: all]
  --source TEXT                   Configured source name (repeatable). Accepts
                                  'last' for the previously synced source.
  --format [markdown|html]        Output format for rendering (markdown or
                                  html)  [default: html]
  --watch                         Watch sources for changes and run
                                  continuously
  --notify                        Desktop notification on conversation changes
                                  (requires --watch)
  --exec TEXT                     Execute command on conversation changes
                                  (requires --watch)
  --webhook TEXT                  Call webhook URL on conversation changes
                                  (requires --watch)
  --reparse                       Force re-parsing of all raw conversations
                                  (clears parse tracking)
  -h, --help                      Show this message and exit.
