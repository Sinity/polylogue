Usage: polylogue query-completions [OPTIONS]

  List structured query-completion candidates.

Options:
  --kind [field|structural-unit|structural-field|count-operator|date-operator|action]
                                  Candidate kind to inspect.  [required]
  --incomplete TEXT               Current incomplete token.  [default: ""]
  --unit TEXT                     Structural unit for structural-field
                                  completion.
  --field TEXT                    Field name for count/date operator
                                  completion.
  -f, --format [plain|json]       [default: plain]
  -h, --help                      Show this message and exit.
