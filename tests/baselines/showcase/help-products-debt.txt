Usage: polylogue products debt [OPTIONS]

  List archive debt and maintenance readiness products.

Options:
  --category TEXT    Only this maintenance category
  --only-actionable  Only debt entries with pending issues
  --limit INTEGER    Maximum rows  [default: 50]
  --offset INTEGER   Start offset  [default: 0]
  --json             Output as JSON
  --format [json]    Output format
  -h, --help         Show this message and exit.
