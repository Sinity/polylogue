Usage: polylogue backup [OPTIONS]

  Back up the Polylogue archive durability tiers.

Options:
  --output-dir PATH  Directory to write the backup into.  [required]
  --check            Verify backup prerequisites without creating a backup.
  --include-blobs    Also back up the blob store directory for legacy
                     archives.
  --verify           Restore the finished backup into a scratch directory and
                     run smoke checks.
  -h, --help         Show this message and exit.
