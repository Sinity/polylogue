Usage: polylogue insights export [OPTIONS]

  Export versioned archive-insight bundles.

Options:
  --out PATH               Output bundle directory.  [required]
  --insight TEXT           Insight to include. Defaults to all exportable
                           insights.
  --provider TEXT          Limit supported insights to one provider.
  --since TEXT             Limit supported insights to rows at/after this
                           timestamp or date.
  --until TEXT             Limit supported insights to rows at/before this
                           timestamp or date.
  --bundle-format [jsonl]  [default: jsonl]
  --format [json]          Output format.
  --overwrite              Replace an existing bundle directory after writing
                           a complete new one.
  -h, --help               Show this message and exit.
