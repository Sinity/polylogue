# Scale and corpus

Archive size is an activation condition, but current scale smokes have weak
oracles (`0 <= hits <= limit`, nonempty bounded lists). The benchmark fixture
manually constructs storage rows, bypasses provider parsing, and emits tool uses
without the tool-result relations needed to activate known failures.

## One generator authority

Assume `polylogue-1xc.14.1` has already extended schema-derived generation with
versioned package/archive profiles, bounded mergeable sketches, correlated
variants, tool pairing states, lineage, selectivity, active growth,
convergence, named scale tiers, and C-03. This packet consumes those artifacts;
it does not extend `SyntheticCorpus`, define another relation-profile format,
or retain another archive sample.

Build immutable seeded archives through acquire → parse → materialize → index
using the upstream workload/profile/build/archive identity and shared
`polylogue-1xc.14` receipt. The cache adapter may extend identity with DDL,
parser, and materializer digests; it may not mint a competing workload or
resource-receipt identity. Seed alone remains insufficient.

Map consumers onto the realized named tiers and canaries. Keep tiny curated
prose only where ranking/rendering semantics require known text. Independent
expected facts attach to a canary; they are not another workload profile.

Use tmpfs for ordinary multi-GB immutable fixtures where measured feasible;
build once under a lock, publish atomically, then reflink/copy per mutating test.
Keep exceptional huge/I/O-realistic tests outside tmpfs. Measure concurrency,
RSS, rebuild time, and invalidation rather than encoding filesystem folklore.

Begin with the realized C-03 exact-session actions canary. Replace the six
current scale smokes only after exact membership/count laws and work-growth
receipts run on shared workload identities, and only after every benchmark
consumer of the old seeder is migrated.

The pre-dispatch dossier must identify the merged C-03 test/receipt path, named
scale-tier API, privacy-reviewed profile artifact, and shared receipt symbols.
Missing names block execution rather than authorizing a local replacement.
